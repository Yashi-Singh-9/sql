# Data Manipulation Language (DML)

**Data Manipulation Language (DML)** is a subset of SQL used to manage and interact with data stored in database objects. DML commands operate on the **data itself**, not on the database structure.

## DML Commands

- **SELECT** – Retrieves data from one or more tables  
- **INSERT** – Adds new records to a table  
- **UPDATE** – Modifies existing records in a table  
- **DELETE** – Removes records from a table  

## Purpose of DML

DML statements enable users to:
- Retrieve and analyze data
- Add new information
- Update existing data
- Remove outdated or incorrect records

## Summary

DML commands are essential for **day-to-day database operations**, data analysis, and maintaining the accuracy, consistency, and relevance of data within a database system.
