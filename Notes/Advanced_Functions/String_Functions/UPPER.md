# UPPER

`UPPER()` is a **string function** in SQL used to convert all alphabetic characters in a specified string to **uppercase**. Non-alphabetic characters remain unchanged.

## Key Characteristics

- Converts text to **uppercase**
- Works on alphabetic characters only
- Leaves numbers and special characters unchanged

## Common Use Cases

The `UPPER()` function is often used for:
- **Data normalization**
- Performing **case-insensitive comparisons**
- Formatting output in `SELECT` statements
- Applying case-insensitive conditions in `WHERE` clauses

## Related Functions

- `LOWER()` – converts all characters in a string to lowercase

## Important Considerations

When working with **international character sets**, the behavior of `UPPER()` may vary depending on locale and collation settings in the database.

## Summary

`UPPER()` is a simple yet powerful function for standardizing text data, improving consistency, and enabling reliable case-insensitive operations in SQL queries.
