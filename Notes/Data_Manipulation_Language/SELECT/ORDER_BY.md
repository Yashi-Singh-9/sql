# ORDER BY

The `ORDER BY` clause in SQL is used to sort the result set of a query based on one or more columns.

## Key Features of ORDER BY

- Sorts data in **ascending order by default**
- Supports **descending order** using the `DESC` keyword
- Can sort by:
  - Numeric values
  - Date and time values
  - Text values
- Allows sorting by **multiple columns**, each with its own sort direction

## Usage

When sorting by multiple columns, the data is ordered by the first column, and then further sorted by the subsequent columns as needed.

## Purpose

The `ORDER BY` clause is essential for presenting data in a meaningful sequence, such as:
- Displaying the most recent records first using timestamps
- Sorting records alphabetically by name

## Summary

`ORDER BY` enhances the readability and usability of query results by organizing data into a clear and logical order.
