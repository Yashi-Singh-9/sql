# GROUP BY

`GROUP BY` is an SQL clause used in `SELECT` statements to arrange identical data into groups. It is commonly used together with **aggregate functions** to perform calculations on each group of rows.

## Key Uses of GROUP BY

- Groups rows that have the same values in specified columns  
- Works with aggregate functions such as:
  - `COUNT`
  - `SUM`
  - `AVG`
  - `MIN`
  - `MAX`
- Enables analysis of data at a **higher level of granularity**

## Purpose

The `GROUP BY` clause collects data across multiple records and groups the results by one or more columns. It is fundamental for:
- Generating **summary reports**
- Performing **data analysis**
- Creating meaningful **aggregations** in relational databases

## Summary

`GROUP BY` helps transform detailed data into summarized insights, making it an essential tool for reporting and analytical queries.
