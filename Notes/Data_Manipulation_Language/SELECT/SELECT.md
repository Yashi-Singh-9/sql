# SELECT

`SELECT` is one of the most fundamental SQL commands, used to retrieve data from one or more tables in a database.

## Key Capabilities of SELECT

- Specify **which columns** to retrieve  
- Fetch data from **one or multiple tables**  
- Apply **filtering conditions** using `WHERE`  
- **Sort results** using `ORDER BY`  
- Perform **aggregations** such as `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX`  
- Use **joins** to combine data from related tables  
- Support **subqueries** for advanced data retrieval  

## Purpose

The `SELECT` statement is highly versatile and essential for **data querying, reporting, and analysis** in relational databases.
