# MAX

`MAX` is an **aggregate function** in SQL that returns the **highest value** from a set of values in a specified column. It can be applied to **numeric**, **date**, or **string** data types.

## Key Features

- Returns the **maximum value** in a column  
- Works with numbers, dates, and strings  
- Frequently used with the `GROUP BY` clause to determine the highest value within each group

## Common Use Cases

The `MAX` function is useful for:
- Finding the **highest salary** in a table  
- Identifying the **most recent date** in a dataset  
- Retrieving the **alphabetically last name** from a list of values

## Summary

The `MAX` function is a valuable SQL tool for data analysis, enabling users to easily identify the largest or most recent values in a dataset or within grouped records.
