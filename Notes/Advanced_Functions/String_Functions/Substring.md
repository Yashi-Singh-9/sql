# SUBSTRING

`SUBSTRING` is a **string function** in SQL used to extract a specific portion of a string. It allows you to define the **starting position** and the **length** of the text to be retrieved.

## Key Characteristics

- Extracts part of a string based on position
- Requires a **starting index** and a **length**
- Works consistently across most SQL databases (with minor syntax variations)

## Common Use Cases

The `SUBSTRING` function is useful for:
- **Data manipulation**
- Parsing structured text
- Formatting and transforming string values
- Extracting meaningful segments from larger strings

## Syntax Considerations

While the exact syntax may differ slightly between database systems, the core concept of extracting a portion of a string remains the same.

## Summary

`SUBSTRING` is a versatile and essential SQL function for working with text data, enabling precise extraction and transformation of string values.
