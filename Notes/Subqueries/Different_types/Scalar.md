# Scalar

A **scalar** value is a single data item, as opposed to a set or collection of values. In SQL, scalar concepts are commonly used in both **subqueries** and **functions**.

## Scalar Subqueries

A **scalar subquery** is a subquery that returns:
- Exactly **one row**
- Exactly **one column**

Scalar subqueries are often used in:
- `SELECT` statements  
- `WHERE` clauses  
- Expressions and calculations  

## Scalar Functions

**Scalar functions** in SQL operate on input values and return a **single value**. These functions are commonly used for:
- Data transformation  
- Calculations  
- Formatting results  

## Importance of Scalar Concepts

Understanding scalar values and scalar subqueries helps:
- Write **precise and efficient** SQL queries  
- Perform accurate **calculations and comparisons**
- Use SQL expressions more effectively  

## Summary

Scalar concepts are fundamental in SQL, representing single values returned by subqueries or functions and playing a key role in building efficient and expressive queries.
