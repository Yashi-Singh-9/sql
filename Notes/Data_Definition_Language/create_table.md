# CREATE TABLE

`CREATE TABLE` is an SQL statement used to define and create a new table in a database. It specifies the table name, column names, data types, and optional constraints.

## Key Components

- **Table name** – Identifies the table in the database  
- **Column names** – Define the fields within the table  
- **Data types** – Specify the type of data each column can store  
- **Constraints** (optional), such as:
  - Primary keys
  - Foreign keys
  - Default values
  - Unique and NOT NULL constraints

## Purpose

The `CREATE TABLE` statement establishes the structure of a table, defining how data will be stored and organized. It is a fundamental command in database management, essential for setting up the database schema and preparing it to store data.
