# LOWER

`LOWER()` is a **string function** in SQL that converts all alphabetic characters in a specified string to **lowercase**. Non-alphabetic characters and numbers remain unchanged.

## Key Characteristics

- Converts text to **lowercase**
- Accepts a single string argument
- Does not affect numbers or special characters

## Common Use Cases

The `LOWER()` function is commonly used for:
- **Standardizing data**
- Performing **case-insensitive comparisons**
- Formatting output in `SELECT` statements
- **Data cleaning** and normalization
- Improving consistency across different systems

## Related Functions

- `UPPER()` – converts all characters in a string to uppercase

## Summary

`LOWER()` helps ensure consistent text representation and enables reliable, case-insensitive operations in SQL queries, making it an essential tool for data processing and cleanup.
