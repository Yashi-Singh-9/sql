# LENGTH

`LENGTH` is a **string function** in SQL that returns the **number of characters** in a string. It is commonly used to measure the size of text data.

## Key Characteristics

- Returns the **character count** of a string  
- Useful for analyzing and validating text data  
- May behave differently with **multi-byte character sets** depending on the database system  

## Variations Across Databases

While most SQL dialects support `LENGTH`, some databases use alternative function names:
- `LEN` (SQL Server)
- `CHAR_LENGTH` (standard SQL and some databases)

## Common Use Cases

The `LENGTH` function is useful for:
- Enforcing **character limits**
- Formatting and splitting strings
- Identifying **anomalies** in text data
- Validating input lengths

## Summary

`LENGTH` is an essential SQL function for working with string data, helping ensure consistency, accuracy, and proper formatting across databases.
