# GROUP BY

`GROUP BY` is an SQL clause used in `SELECT` statements to arrange identical data into groups. It is commonly used with **aggregate functions** to perform calculations on each group of rows.

## Key Features

- Groups rows that share the same values in specified columns  
- Works with aggregate functions such as:
  - `COUNT`
  - `SUM`
  - `AVG`
  - `MIN`
  - `MAX`
- Allows grouping by one or more columns

## How It Works

The `GROUP BY` clause collects data from multiple records and groups the results based on one or more columns. Aggregate functions are then applied to each group, producing summarized results.

## Use Cases

`GROUP BY` is fundamental for:
- Generating **summary reports**
- Performing **data analysis**
- Creating meaningful **aggregations** of data in relational databases

## Summary

The `GROUP BY` clause enables higher-level data analysis by organizing detailed records into grouped summaries, making it an essential tool in SQL querying.
