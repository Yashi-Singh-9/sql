# ALTER TABLE

The `ALTER TABLE` statement in SQL is used to modify the structure of an existing table without deleting the table itself.

## Common Uses of ALTER TABLE

- **Add new columns** to a table  
- **Drop existing columns**  
- **Modify column definitions**, such as changing data types or sizes  
- **Set or change default values** for columns  
- **Add or remove constraints**, including:
  - Primary keys
  - Foreign keys
  - Unique constraints

## Summary

The `ALTER TABLE` command provides flexibility in database design by allowing structural changes to tables as application requirements evolve.
