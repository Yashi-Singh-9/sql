# Table

A **table** is a fundamental structure used to organize data in a relational database. It stores data in a structured format consisting of **rows (records)** and **columns (fields)**.

## Key Characteristics

- Represents a collection of **related data entries**
- Organized into rows and columns
- Defines a **schema**, including:
  - Column names
  - Data types
  - Constraints

## Role in SQL Databases

Tables are the primary objects used to:
- **Store data**
- **Retrieve data** using SQL queries
- Enforce **data integrity** through constraints

## Importance of Tables

Understanding table structure is essential for:
- Effective **database design**
- Writing efficient and accurate **SQL queries**
- Maintaining data consistency and reliability

## Summary

Tables are the core building blocks of relational databases, providing the structure needed to store, manage, and query data efficiently.
