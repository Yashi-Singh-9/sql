# MIN

`MIN` is an **aggregate function** in SQL that returns the **lowest value** from a set of values in a specified column. It can be applied to **numeric**, **date**, or **string** data types.

## Key Features

- Returns the **minimum value** in a column
- Works with numbers, dates, and strings
- Commonly used with the `GROUP BY` clause to find the minimum value within each group

## Common Use Cases

The `MIN` function is useful for:
- Finding the **lowest price** in a product list
- Identifying the **earliest date** in a dataset
- Retrieving the **alphabetically first name** in a table

## Summary

The `MIN` function is a powerful tool for data analysis, helping extract the smallest value from a dataset or within grouped records.
