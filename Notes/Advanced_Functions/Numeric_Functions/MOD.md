# MOD

`MOD` is a **numeric function** in SQL that calculates the **remainder** when one number is divided by another.

## How MOD Works

The function takes **two arguments**:
1. The **dividend**
2. The **divisor**

It returns the remainder of the division operation.

### Example
- `MOD(10, 3)` → `1`

## Common Use Cases

The `MOD` function is useful for:
- Checking **odd or even** numbers
- Implementing **cyclic or repeating patterns**
- Distributing data evenly across groups
- Performing modular arithmetic in calculations

## Cross-Database Considerations

- Many SQL databases support the `MOD()` function
- Some databases use the `%` (modulus) operator instead of `MOD`
- Syntax and behavior may vary slightly across SQL dialects

## Summary

The `MOD` function is a simple yet powerful tool for remainder-based calculations in SQL, enabling effective handling of cycles, patterns, and numeric conditions.
