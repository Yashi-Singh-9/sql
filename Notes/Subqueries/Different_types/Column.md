# Column

In SQL, a **column** is used to categorize and store data within a table. Each column represents a specific attribute and holds values of a defined data type for all rows in the table.

## Key Characteristics

- Stores a **specific type of data** (e.g., integers, strings, booleans)  
- Defined with a **data type** that controls what values it can hold  
- Applies consistently across all rows in the table  

## Importance of Column Design

Choosing the correct column type and size helps to:
- Maintain **data integrity**
- Prevent invalid data entry
- **Optimize database performance**
- Improve storage efficiency

## Summary

Columns are a fundamental part of SQL tables, providing structure and organization to data. Proper column definition is essential for building efficient, reliable, and well-designed databases.
