# REPLACE

`REPLACE` is a **string function** in SQL used to substitute all occurrences of a specified substring within a string with a new substring.

## How It Works

The `REPLACE` function takes **three arguments**:
1. The original string  
2. The substring to be replaced  
3. The substring to replace it with  

If the specified substring is found, all instances are replaced. If it is not found, the original string is returned unchanged.

## Common Use Cases

The `REPLACE` function is especially useful for:
- **Data cleaning**
- Correcting typos
- Standardizing text formats
- Replacing outdated or obsolete values

## Key Characteristics

- Replaces **all occurrences** of the target substring  
- Does not modify the original data unless used in an update operation  
- Works on string-based data

## Summary

`REPLACE` is a powerful and practical SQL function for modifying text data, making it an essential tool for maintaining clean and consistent datasets.
