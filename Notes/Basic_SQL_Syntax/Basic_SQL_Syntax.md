# Basic SQL Syntax

Basic SQL syntax consists of straightforward commands that allow users to interact with a relational database. These commands form the foundation of SQL and enable efficient data manipulation and retrieval.

## Core SQL Commands

### SELECT
Used to query and retrieve data from one or more tables.

### INSERT INTO
Used to add new records (rows) into a table.

### UPDATE
Used to modify existing data within a table.

### DELETE
Used to remove records from a table.

## Query Clauses

### WHERE
Filters records based on specified conditions.

### ORDER BY
Sorts query results in ascending or descending order.

### JOIN
Combines rows from two or more tables based on a related column.

## Summary

These core commands and clauses form the foundation of SQL, allowing users to efficiently query, insert, update, and delete data within a relational database.
