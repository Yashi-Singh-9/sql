# AVG

The `AVG()` function in SQL is an **aggregate function** used to calculate the **average value** of a numeric column.

## Key Features of AVG

- Computes the **sum of all values** in a column divided by the **number of values**  
- Ignores **NULL values** in the calculation  
- Often used with the `GROUP BY` clause to calculate averages for different groups  

## Use Cases

- Calculating **average scores**, **sales**, or **temperatures**  
- Generating **summary statistics** for reports  
- Analyzing **performance trends** within datasets  

## Summary

`AVG()` is essential for calculating averages, providing insights and statistical summaries from numeric data in SQL tables.
