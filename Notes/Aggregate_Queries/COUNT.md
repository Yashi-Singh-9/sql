# COUNT

`COUNT` is an **aggregate function** in SQL that returns the number of rows that match specified criteria.

## Key Features of COUNT

- Can count **all rows** in a table  
- Can count **non-NULL values** in a specific column  
- Can be combined with a `WHERE` clause to count **rows that meet certain conditions**  

## Use Cases

- **Data analysis** – determining the number of records  
- **Reporting** – summarizing dataset sizes or value frequencies  
- **Performance optimization** – analyzing table or query sizes  

## Summary

The `COUNT` function is essential for understanding dataset sizes, summarizing information, and generating meaningful statistics in SQL.
