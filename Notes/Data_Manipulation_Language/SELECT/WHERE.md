# WHERE

The `WHERE` clause in SQL is used to **filter records** based on specified conditions. Only the rows that satisfy the given condition are returned or affected by the query.

## Purpose of WHERE

- Filters records to fetch **only the necessary data**  
- Applies conditions using comparison and logical operators  
- Helps improve query efficiency and accuracy  

## Usage of WHERE

The `WHERE` clause is commonly used with:
- **SELECT** – to retrieve specific records  
- **UPDATE** – to modify selected records  
- **DELETE** – to remove specific records  

## Summary

The `WHERE` clause is essential for controlling which records are selected or modified in SQL queries, ensuring precise and efficient data operations.
