# CONCAT

`CONCAT` is a **string function** in SQL used to combine two or more strings into a single string. It joins the input strings in the order they are provided and returns the resulting combined value.

## Key Characteristics

- Accepts **multiple string arguments**
- Returns a **single concatenated string**
- Preserves the order of the input values

## Common Use Cases

The `CONCAT` function is commonly used to:
- Merge data from **multiple columns**
- Create **custom-formatted output**
- Build readable text values (e.g., full names)
- Generate **dynamic SQL expressions**

## Summary

`CONCAT` is a simple yet powerful SQL function for combining text values, making it essential for formatting and presenting string-based data effectively.
