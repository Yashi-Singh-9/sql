# SUM

`SUM` is an **aggregate function** in SQL used to calculate the total of a set of numeric values.

## Key Features of SUM

- Calculates the **sum of values** in a numeric column  
- Often used with the `GROUP BY` clause to compute totals for **different categories or groups**  
- Ignores **NULL values** in the calculation  
- Can be combined with other aggregate functions for **complex analysis**  

## Use Cases

- **Financial calculations** (e.g., total sales, expenses)  
- **Statistical analysis** (e.g., total scores, quantities)  
- **Summary reports** from database tables  

## Summary

The `SUM` function is essential for aggregating numeric data, enabling effective analysis and reporting in SQL.
